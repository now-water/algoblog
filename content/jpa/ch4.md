# 엔티티 매핑

대표적인 매핑 어노테이션.

XML에 기입해도 되지만 어노테이션 방식이 좀 더 쉽고 직관적이다

| 매핑               | 어노테이션              |
| ------------------ | ----------------------- |
| 객체와 테이블 매핑 | @Entity, @Table         |
| 기본 키 매핑       | @Id                     |
| 필드와 컬럼 매핑   | @Column                 |
| 연관관계 매핑      | @ManyToOne, @JoinColumn |
